# TikTok Embed Fix

Fixes TikTok embeds in Discord by injecting an fxTikTok link while keeping the original TikTok URL.

## Example

Original message:

https://vt.tiktok.com/ZSuPgwwBC/

Sent message:

TikTok
https://fixtiktok.com/ZSuPgwwBC/
https://vt.tiktok.com/ZSuPgwwBC/

Discord shows the embed from fxTikTok but users still see the original link.

## Supported links

- tiktok.com
- vt.tiktok.com
- vm.tiktok.com

## Installation

1. Upload the plugin to your Vendetta / Ketutweak plugin folder
2. Enable the plugin
3. Send a TikTok link
